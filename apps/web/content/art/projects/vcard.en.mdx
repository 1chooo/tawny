**VCard** is a portfolio template I developed and open-sourced for developers who want a fast, polished personal site without starting from scratch.

## Highlights

- **Next.js** App Router with server components by default
- **Supabase** for optional dynamic content and auth patterns
- **99+ Lighthouse** performance out of the box — careful image handling, font loading, and minimal client JS
- **MDX-ready** structure for a blog or project showcase

The template has grown to 300+ GitHub stars and is used by developers across several countries. I maintain it as a reference implementation for how I think portfolio sites should feel: fast, accessible, and easy to customize.

View the [live demo](https://vcard.1chooo.com) or fork it on [GitHub](https://github.com/1chooo/vcard).
