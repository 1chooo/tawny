export const postMeta = {
  title: "打造這本日誌",
  date: "2026-06-06",
  description:
    "混合式指南——這個網站為何存在、用什麼搭建，以及如何一步步重建它。",
  tags: ["guide", "nextjs", "mdx", "build"],
};

這本日誌是一個安放長文的地方——以排版為先、支援雙語，並刻意分成兩種氛圍：**首頁的粗獷 bento 網格**，以及**文章頁的編輯式閱讀體驗**。

首頁用結構說話；文章用文字呼吸。這種對比是刻意的。這篇指南會同時說明「為什麼」與「怎麼做」。

## 逐步搭建

<Steps>
  <Step title="建立 Next.js + Tailwind v4 專案">
    以 <InlineCode>create-next-app</InlineCode> 建立 App Router + TypeScript 專案。透過 <InlineCode>@tailwindcss/postcss</InlineCode> 加入 Tailwind v4，並在 <InlineCode>globals.css</InlineCode> 定義語意化色彩 token——文章用 <InlineCode>paper</InlineCode>、<InlineCode>ink</InlineCode>、<InlineCode>accent</InlineCode>；首頁 bento 用 <InlineCode>bento-bg</InlineCode>、<InlineCode>bento-ink</InlineCode>。
  </Step>
  <Step title="設定 MDX 與 remark-gfm">
    安裝 <InlineCode>@next/mdx</InlineCode>、<InlineCode>@mdx-js/react</InlineCode> 與 <InlineCode>remark-gfm</InlineCode>。以 <InlineCode>createMDX</InlineCode> 包裝 Next 設定，並將 <InlineCode>mdx</InlineCode> 加入 <InlineCode>pageExtensions</InlineCode>。在 <InlineCode>mdx-components.tsx</InlineCode> 統一設定 markdown 元素的樣式。
  </Step>
  <Step title="加入 next-intl 雙語路由">
    設定 <InlineCode>en</InlineCode>、<InlineCode>zh</InlineCode> 語系，並使用 <InlineCode>localePrefix: 'always'</InlineCode>，讓每個 URL 都帶語系前綴。UI 字串放在 <InlineCode>messages/en.json</InlineCode> 與 <InlineCode>messages/zh.json</InlineCode>。全站連結使用 <InlineCode>@/i18n/navigation</InlineCode> 以保持語系一致。
  </Step>
  <Step title="建立文章（MDX + 註冊表）">
    每篇文章放在 <InlineCode>content/posts/</InlineCode>。匯出 <InlineCode>postMeta</InlineCode> 物件，包含 title、date、description 與可選的 tags。在 <InlineCode>lib/posts.ts</InlineCode> 手動註冊 import——這種方式型別明確，建置結果可預期。
  </Step>
  <Step title="以 prose-blog 與 bento 網格完成樣式">
    文章使用 <InlineCode>@tailwindcss/typography</InlineCode> 的 <InlineCode>prose-blog</InlineCode> 工具類。首頁以 12 欄 bento 網格組合圖片、標籤與最新文章輪播——除輪播控制外，皆由 Server Component 渲染。
  </Step>
</Steps>

## 動手試試

新文章的基本結構如下：

<CodeBlock
  title="content/posts/my-post.mdx"
  language="mdx"
  code={`export const postMeta = {
  title: "我的第一篇筆記",
  date: "2026-06-06",
  description: "供列表與 SEO 使用的簡短摘要。",
  tags: ["journal"],
};

你的 MDX 內容從這裡開始。`}
/>

在註冊表中加入這篇文章：

<CodeBlock
  title="lib/posts.ts"
  language="typescript"
  code={`import MyPost, { postMeta as metaMyPost } from "@/content/posts/my-post.mdx";

const ENTRIES: PostEntry[] = [
  {
    slug: "my-post",
    locales: { en: { meta: metaMyPost, MDXContent: MyPost } },
  },
  // ...既有項目
];`}
/>

啟動開發伺服器：

<CodeBlock
  title="終端機"

/>
