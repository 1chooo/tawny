**VCard** 是我開發並開源的個人作品集模板，給不想從零開始、又想要快速、精緻網站的開發者。

## 特色

- **Next.js** App Router，預設使用 server components
- **Supabase** 作為可選的動態內容與驗證模式
- 開箱即達 **99+ Lighthouse**——謹慎處理圖片、字型載入與極少 client JS
- **MDX** 友善結構，適合部落格或專案展示

模板在 GitHub 上已超過 300 stars，被多國開發者採用。我持續維護它，作為我心目中作品集網站該有的樣子：快速、可及、易於客製。

查看 [線上展示](https://vcard.1chooo.com) 或在 [GitHub](https://github.com/1chooo/vcard) fork。
