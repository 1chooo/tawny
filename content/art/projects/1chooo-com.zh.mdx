[1chooo.com](https://1chooo.com) 是我在網路上的主要據點——一個流暢、極簡的空間，承載筆記、專案與個人服務。

## 技術棧

- **Next.js v16** — App Router 與 React Server Components
- **Tailwind CSS v4** — 樣式與設計代幣
- **Motion** — 細膩而有目的的動畫
- **MDX** — 長文內容與自訂短碼
- **Cloudflare R2** 與 **Supabase** — 媒體與資料

## 內容

除了筆記與專案頁面，網站還有一組日常工具——短網址、實用小程式與實驗。一切為快速載入與安靜的閱讀體驗而優化。
