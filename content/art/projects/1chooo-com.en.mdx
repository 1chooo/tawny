I built [1chooo.com](https://1chooo.com) as my primary home on the web — a seamless, minimal experience for notes, projects, and personal services.

## Stack

- **Next.js v16** with the App Router and React Server Components
- **Tailwind CSS v4** for styling and design tokens
- **Motion** for subtle, purposeful animation
- **MDX** for long-form content with custom shortcodes
- **Cloudflare R2** and **Supabase** for media and data

## What it includes

Beyond the journal and project pages, the site hosts a suite of everyday tools — link shorteners, utilities, and experiments I reach for regularly. Everything is optimized for fast loads and a calm reading experience.

Explore the live site at [1chooo.com](https://1chooo.com) or browse the source on [GitHub](https://github.com/1chooo/1chooo.com).
